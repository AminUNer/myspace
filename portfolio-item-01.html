Description

Projets Intégrés durant la troisiéme et la quatriéme année de spécialité en cycle ingénieur .

Basée sur la méthodologie agile, ce projet se compose de 3 Sprints, le premier est la développement du backend JAVA EE ainsi que l'exposition des web services REST API, le deuxiéme concerne le développement du dashboard adminstratif en utilisant ASP.NET (Database first), finalement, le dernier sprint Angular, tout ce qui concerne le développement de la partie front et des interfaces utilisateur et la consommation des web services API REST .

Etablissement

ESPRIT

Technologies

  • Ionic
  • JAVA EE
  • REST API
  • HTML 5
  • CSS 3
  • Responsive Design
